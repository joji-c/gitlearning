configeration:
to start with git:                                       (if no verion download git from browser)
to see git version                                       -> git --version                                  
to see the git config details                            -> git config --list
to see the user and his changes made use                 -> git log
to add or change username                                -> git config --global user.name "username"
to add or change email                                   -> git config --global user.email "user@gmail.com"
to check if git is upto date or untracked use            -> git status
add file to git repository                               -> git init
git_add[adding chnages to SA]                            -> git add -A 
git_commit[adding staged files to LR]                    -> git commit -m "message"
get RR details                                           -> git remote -v
to add a RR                                              -> git remote add name path
to know on which branch and its details                  -> git branch
to push from LR to RR                                    -> git push name branch_name
